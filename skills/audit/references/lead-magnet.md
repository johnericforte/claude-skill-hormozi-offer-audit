# Lead Magnet Reference Card

> Source: Alex Hormozi, _$100M Leads_ (2023). Framework cited descriptively under nominative fair use.

## Definition

A lead magnet is a complete solution to a narrow problem, given away to attract the right buyer to a paid offer. Hormozi covers lead magnets in $100M Leads as one of the four primary lead-getting methods (alongside warm outreach, cold outreach, and paid ads).

## Why this skill audits the Quality Bar, not a step-by-step build

Published "step-by-step" lead magnet builds attributed to Hormozi online (commonly framed as a "7-step build") are typically third-party interpretations, not direct excerpts from $100M Leads. To stay accurate and avoid misattribution, this skill does not enforce a specific numbered build process. The canonical build process lives in the book (Section II: Lead Magnets); this skill audits against the Quality Bar that any lead magnet must clear before shipping.

If you want a step-by-step build process derived from $100M Leads, the safest path is to read the book directly. This skill scores existing magnets against the four-item ship test below.

## Quality Bar (4-item ship test)

A lead magnet ships when ALL four are true. If 1+ fails, the magnet leaks leads or attracts the wrong audience.

### 1. Valuable enough someone could pay for it

Audit question: would the buyer pay $X for this if it weren't free?

- If yes: ship.
- If no: it's a brochure, not a magnet. Rebuild.

### 2. Narrow enough to solve in one sitting

Audit question: can the buyer apply it before they close the tab?

- If yes: ship.
- If no: split into multiple magnets, each narrower.

### 3. Easy to consume / fast time-to-value

Audit question: does the reader hit the "ah, that's useful" moment within 15 minutes?

- If yes: ship.
- If no: trim. The most common failure mode is over-explaining the problem before delivering the fix.

### 4. Bridges naturally to the core offer

Audit question: does the magnet make the buyer think "I want more of this from this person"?

- If yes: ship.
- If no: the magnet doesn't qualify the right lead. Either narrow the magnet or change the core offer.

## Audit output format

```
Quality Bar:
  1. Pay for it?              YES
  2. One sitting?             YES
  3. Fast time-to-value?      YES
  4. Bridges to core offer?   NO

Verdict: Do not ship. Quality bar failure on item 4 (no clear bridge to core offer).
```

## Common audit findings

- **Magnet is a "complete guide" without a fix** → fails Quality Bar item 1 and 3. Trim to a single deliverable; remove the explanatory padding.
- **Magnet ends with "thanks for downloading"** → fails item 4. Add a CTA tied to the core offer.
- **Magnet covers a category, not a problem** → fails item 2. Pick one specific problem inside the category.
- **Magnet feels like everything else in the inbox** → fails item 1 (perceived value). Format and content quality both need to exceed competitors' free assets.
